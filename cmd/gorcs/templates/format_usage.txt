Usage: gorcs format [flags...] [files...]

Subcommands:
    help         Print this help message
    usage        Print this usage message

Flags:
    -o, --output string   Output file path (only valid for single input file)
    -f, --force           Force overwrite output file if it exists
    -w, --overwrite       Overwrite input file
    -s, --stdout          Force output to stdout
    --keep-truncated-years Keep truncated years (do not expand to 4 digits)

Positional Arguments:
    files      List of files to process or - for stdin
